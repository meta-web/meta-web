## Boolean property

**`@type: meta/properties/boolean`**

Boolean property

| Property | Type | * | # | Description |
| -------- | ---- | - | - | ----------- |
| style | String | No | No | Defines how to display control - one of (checkbox / switch). Defaults to `checkbox` |
