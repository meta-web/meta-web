## E-mail control
`application/x.meta.control.text.email+json`

Text control with e-mail validation. Inherits from `application/x.meta.control.text+json` control.


| Property name | Type    | Description           | Required |
| ------------- | ------- | --------------------- | -------- |
| showMailto    | boolean | If to show mailto button - defaults to true | No |
