## Boolean control

Boolean input control. Can be displayed as checkbox or switch.


| Property name | Type   | Description           | Required |
| ------------- | ------ | --------------------- | -------- |
| style         | string  | Defines how to display control - one of (checkbox / switch). Defaults to `checkbox` | No |
